Hello , Welcome to my API

This is a simple rest API made using FLASK as backend , tested using unittest module of python , Database used is MongoDB and deployed on ________ .
The API has the following endpoints :

  • /display --> display all the elements in the list
  • /add --> Add an item in todo list
  • /edit/_id --> edit an item in the list upon providing its id
  • /delete/_id --> delete an element in the list upon providing its id

Note: By running test.py, you can run the test cases. The test cases run in a sequential manner, i.e., a piece of data is added to the list, that same piece of data is updated, and then deleted. Therefore, we are dealing with only one piece of data in testing, and there is no need to clear the test database after running all the tests.


Note: By running run.py, you can start the server and test the endpoints on clients such as Postman.
